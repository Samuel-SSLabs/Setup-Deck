O Projeto
O Setup Deck Companion é o centro de controle definitivo para suas controladoras Pulse e Core Deck. Projetado desde o hardware até o código para oferecer máxima performance, baixa latência e uma interface totalmente intuitiva. Altere atalhos, configure macros complexas e gerencie o feedback háptico do motor em tempo real.

Versão atual: v1.2 (Beta)
